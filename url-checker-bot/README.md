# 🔒 بوت فحص الروابط الآمنة

بوت تلكرام ذكي لفحص أمان الروابط باستخدام Python و مكتبة telebot.

## 📋 المميزات

✅ **فحص فوري للروابط** - تحقق من أمان أي رابط في ثوانٍ  
✅ **كشف التهديدات** - التعرف على الروابط الخطيرة والمريبة  
✅ **معلومات تفصيلية** - الحصول على تفاصيل كاملة عن الرابط  
✅ **واجهة سھلة الاستخدام** - تصميم بسيط وسهل  
✅ **دعم Google Safe Browsing** - استخدام قاعدة بيانات جوجل للروابط الآمنة  
✅ **دعم اللغة العربية** - واجهة كاملة باللغة العربية

## 🚀 البدء السريع

### المتطلبات

- Python 3.7 أو أحدث  
- رمز بوت تلكرام (من [@BotFather](https://t.me/BotFather))  
- اتصال بالإنترنت

### التثبيت

1. **استنساخ المستودع:**
```bash
git clone https://github.com/uug405528-dev/Hasen-api.git
cd Hasen-api/url-checker-bot
```

2. **تثبيت المكتبات:**
```bash
pip install -r requirements.txt
```

3. **إعداد ملف البيئة:**
```bash
cp .env.example .env
```

4. **تحديث الرمز في .env:**
```
BOT_TOKEN=YOUR_ACTUAL_BOT_TOKEN_HERE
GOOGLE_API_KEY=YOUR_GOOGLE_API_KEY_HERE (اختياري)
```

5. **تشغيل البوت:**
```bash
python main.py
```

## 📖 طريقة الاستخدام

### الأوامر المتاحة

| الأمر | الوصف |
|------|-------|
| `/start` | ابدأ مع البوت |
| `/help` | عرض المساعدة |
| `/about` | معلومات عن البوت |

### فحص الروابط

ببساطة أرسل أي رابط للبوت:
```
https://google.com
```

البوت سيفحصه ويرسل لك:
- ✅ إذا كان آمن  
- ⚠️ إذا كان مريب  
- ❌ إذا كان خطير

## 🔧 الفحوصات المستخدمة

1. **التحقق من صيغة الرابط** - التأكد من أن الرابط بصيغة صحيحة
2. **فحص الاتصال بالخادم** - محاولة الاتصال بالخادم
3. **التحقق من رمز الحالة HTTP** - فحص حالة الاستجابة
4. **Google Safe Browsing** - فحص قاعدة بيانات جوجل (إن توفر API Key)

## 🛡️ ملاحظات أمنية

⚠️ **مهم جداً:**  
- الفحص قد لا يكون دقيق بنسبة 100%  
- استخدم الحذر مع الروابط المريبة  
- لا تنقر على روابط خطيرة حتى لو كنت متأكداً  
- حافظ على رمز البوت سراً

## 🔐 إنشاء Google Safe Browsing API Key

1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com/)
2. أنشئ مشروع جديد
3. فعّل Safe Browsing API
4. أنشئ API Key
5. أضفه في ملف `.env`
